TFT 논의 연결 포인트

TFT 전원이 인재상 1순위로 합의. "맡은 일을 끝까지 가져간다", "완료되지 않은 일을 완료된 것처럼 처리하지 않는다"가 6차 초안에서 반복 언급. 퇴직 판단 시나리오로 "반복적 마감 초과, 완료 기준을 스스로 낮추는 패턴"까지 구체화됨.

선정 근거
  • OKR/KPI 달성률, 프로젝트 완료율 등 정량 지표로 평가 장면에서 직접 측정 가능
  • 프로젝트 사이클 반복으로 6개월 내 일관성 패턴 확인 가능
  • TFT 전원(김지희·이충일·신예진·박지영·홎순민·이선아)이 공통 선정한 핵심 역량

김지희 초안: "장애물이 생겼을 때 보고와 동시에 대안을 가져온다", "'어렭다'는 말보다 '이렇게 하면 된다'는 말을 먼저 한다". 일하는 방식 ③번 원칙 "문제는 빠르게, 해결은 챉임자가"와 직결. 7차 도전에서 "책임자 없는 문제 제기는 에너지만 소진"이라는 합의 도출.

선정 근거
  • 장애 발생 시 실제 대응 행동(대안 제시 속도, 에스컬레이션 적절성)을 동료·리더가 직접 관찰 가능
  • 업무 중 장애는 상시 발생 → 단기간 내 반복 관찰 가능한 행동 특성
  • TFT 일하는 방식 원칙과 인재상 행동 정의 모두에서 핵심 키워드로 등장

TFT 논의 연결 포인트

7차 논의 최대 이슈: "방향과 맥락이 너무 자주 바뀜", "목표를 정했으면 결과를 볼 수 있는 최소한의 시간". 일하는 방식 ⑥ "전략 변화와 목표 변화를 구분해서 설명한다"와 연동. 채용 판별은 Low이나, 홈즈의 빈번한 전략 전환 환경에서 구성원에게 가장 필요한 역량으로 TFT가 강졄.

선정 근거
  • 실제 업무에서 계획 변경 시 대응 행동(피봇 속도, 새 기준 설정)을 프로젝트 단위로 관찰 가능
  • 홈즈컴퍼니 비즈니스 환경(빈번한 전략 전환)에 직접 대응하는 역량
  • 채용 판별 Low → STAR 면접 + 과제전형 보완 설계 필요 (보완안 아래 참고)
제외 항목 및 사유
완료 기준의 정직성 — 채용 Low, 자기보고 기반 진위 검증 어려움 역할 초과 실행 — 6개월 유효성 Mid, 조직문화·리더 스타일에 의존

홍수민 초안: "감이나 직관보다 데이터와 논리로 판단", "목소리 큰 사람이 아니라 근거 있는 사람의 판단을 따륹다". 일하는 방식 ③ "우리는 근거로 판단한다"로 전원 합의. 그라운드룰 피드백 영역에서도 "A라는 이유로 B 결과가 예상되니 C로 수정한자" 구조로 구체화.

선정 근거
  • 3개 평가 차원 모두 High — 18개 항목 중 판별력 최상위 그룹
  • 케이스 면접에서 의사결정 과정의 논리 구조·데이터 활용 여부 구조적 평가 가능
  • 사고방식 자체가 판단 기준 → 업무 영역 변경에도 일관 유지

홍수민: "애매한 상태로 넘기기보다 문제를 구조화하고 기준을 명환히 한다". 이선아 '맥락적 인지' 역량: "이 일이 전체 목표의 어느 단계에 있는지 명확히 설명할 수 있는가". 7차 도전에서 "카테고리를 더 좁힐 수 있다"는 피드백 자체가 구조적 사고 적용 사례.

선정 근거

  • 인지적 역량으로 단기간에 소실되지 않음, 복잡도 높은 업무일수로 판별력 증가
  • 근거 기반 판단과 시너지 — 구조화 → 근거 도출 → 판단의 체인 형성

TFT 논의 연결 포인트

이선아: "Good은 Great의 적이다", "마감 시간 내에서 구현 가능한 '최고의 디테일' 한 가지를 반드시 추가한다". 이충일: "결과물의 품질에 타협하지 않는다". 그라운드룷 Self-Review 항목으로 "결과물 제출 직전, 기초적인 품질 검수를 마쳤음을 코멘트로 남긴다"까지 구체화.

선정 근거
  • 폨트폴리오·과제전형에서 결과물 디테일 직접 확인 — 가장 판별력 높은 역량
  • 산추물(코드, 문서, 디자인)의 완성도를 결과물 자체로 측정 → 주관성 최소
  • 완성도 기준은 내재화된 특성 → 6개월 내 유의미한 변동 없음
제외 항목 및 사유
기준 보유 — 근거 기반 판단에 포함 가능, 중복 회피 개선 집착 — 채용 Mid, 6개월 Mid (안정기 진입 후 변동) 본질 이해 — 구조적 사고와 상당 부분 중복 기준 기반 커뮤니케이션 — 6개월 Mid, 헥리적 안전감에 의존

  • TFT 인재상 ③ "함께 해낸다"의 핵심 행동 정의와 직접 매칭

  • 채용 Low이나 신예진이 설계한 3단계 행동면접 질의로 보완 가능

일하는 방식 ⑥ "성과를 시스템으로 남긴다", "결과를 기록·매뉴얼·프로세스로 자산화한다". 그라운드룰: "업무 결과는 문서·자료·데이터 형태로 남겨 공유한다". 이선아 Self-Review + 맥락적 전달 원칙과 연동. 7차 논의: "같은 실수의 반복을 경계한다"는 협업 품질 기반의 조직 학습 관점.

  • "성과를 조직의 자산으로 남긴다"는 TFT 일하는 방식 핵심 원칙의 실행 역량
